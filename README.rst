Movie_utill
===========

Movie_utill provides handling *.mp4[movie-file]. For example, when the same movie files are in your direcory, Movie_utill etiher of one of them. Source Code is C++ with OpenCV library.


Build
-----
::
   
   cd build
   cmake ..
   make

How to use
----------
::
   
   cd your_directory
   mu --same .
